# Contracts: Home Landing Page

This feature has no external API contracts. The home landing page is a
purely client-side static page with no backend API calls.

## Future Contracts

When the workout logging feature is implemented, the following contract
will be needed:

- **POST /api/workouts/start** — Accepts a workout type and creates a
  new workout session. This endpoint does not exist yet and is out of
  scope for this feature.
